# Aaargh!

*I feel lucky*, for [The Pirate Bay](https://thepiratebay.se/). A micro HTTP service,
for picking the best torrent available for download, by searching TPB.

### Install

```
git clone https://github.com/mikelaew/aaargh.git`
cd aaargh
npm install
```

## Run tests

```
make test
```
